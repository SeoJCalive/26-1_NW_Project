# NW 프로젝트 가이드

이 폴더는 NW 프로젝트를 실행하고 관찰하는 사용자를 위한 guide 문서를 둔다.

- [`CONTROL.md`](./CONTROL.md): 실행 명령, viewer/controller 입력, 종료/정리 절차.
- [`노드 자료형.md`](./노드%20자료형.md): 각 node가 주고받는 JSON 자료형.
- [`노드 전달.md`](./노드%20전달.md): 각 node가 서로에게 수행하는 polling, forwarding, ACK, retry, dedup, control/status 작업.

이 폴더의 문서는 root current 문서를 대체하지 않는 reference 자료다. 현재 구현 기준이 바뀌면 root의 `README.md`, `IMPLEMENTATION_SPEC.md`, `AI_IMPLEMENTATION_BRIEF.md`와 함께 갱신한다.
